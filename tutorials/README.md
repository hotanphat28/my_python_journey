# Tutorials

This directory contains introductory Python scripts and exercises designed to practice specific concepts.

## Scripts

- **[args_demo.py](args_demo.py)**: Demonstrates how to handle command-line arguments using `sys.argv`.
- **[calculation.py](calculation.py)**: An interactive script that calculates the total and average of user-inputted scores.
- **[coffee_shop_autobot.py](coffee_shop_autobot.py)**: A fun simulation of a coffee shop ordering system.
- **[contacts.py](contacts.py)**: A simple contact management system that stores data in a JSON file (`contacts.json`). Supports adding, searching, and listing contacts.
- **[get_to_know_you.py](get_to_know_you.py)**: A basic interaction script that asks for your name and birth year to calculate your age.
- **[show_calendar_in_month.py](show_calendar_in_month.py)**: Displays the calendar for a specific month and year using the `calendar` module.
- **[student_object.py](student_object.py)**: Demonstrates Object-Oriented Programming (OOP) basics by creating and manipulating a `Student` class.
