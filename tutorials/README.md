# Tutorials

This directory contains introductory Python scripts and exercises designed to practice specific concepts.

## üöÄ How to Run

The tutorials are organized into a CLI menu system. To access them, run the `main.py` script from this directory:

```bash
python main.py
```

This will launch an interactive menu where you can select which tutorial to run.

## üìÇ Structure

- **`main.py`**: The entry point script that provides a textual menu to run the various tutorial scripts.
- **`scripts/`**: Contains the individual Python scripts for each tutorial.
- **`cs50-*/`**: Standalone Flask applications from CS50 exercises.

## üìú Tutorial Scripts

The following scripts (located in the `scripts/` directory) are available via the menu or direct execution:

- **[args_demo.py](scripts/args_demo.py)**: Demonstrates how to handle command-line arguments using `sys.argv`.
- **[calculation.py](scripts/calculation.py)**: An interactive script that calculates the total and average of user-inputted scores.
- **[coffee_shop_autobot.py](scripts/coffee_shop_autobot.py)**: A fun simulation of a coffee shop ordering system.
- **[contacts.py](scripts/contacts.py)**: A simple contact management system that stores data in a JSON file (`contacts.json`).
- **[get_to_know_you.py](scripts/get_to_know_you.py)**: A basic interaction script that asks for your name and birth year.
- **[show_calendar_in_month.py](scripts/show_calendar_in_month.py)**: Displays the calendar for a specific month and year.
- **[student_object.py](scripts/student_object.py)**: Demonstrates OOP basics with a `Student` class.

## üåê Flask Tutorials (CS50)

- **[CS50 Froshims](cs50-froshims/)**: A sports registration app using Flask and SQLite.
- **[CS50 Greeting](cs50-greeting/)**: A simple greeting app demonstrating GET/POST requests.
